## Installation

Install and configure `git`. After that go to the home folder and clone the project:

`git clone --bare git@github.com:mskalnik/dotfiles.git $HOME/.dotfiles`

Run `setup.sh`:

`chmod +x setup.sh && ./setup.sh`
